# Desafio 01

Encontre o erro rs

## Instalação

Baixe o projeto:

```shell
$ git clone git@github.com:Clyff/desafio-01.git
```

Suba os containers docker:

```shell
$ docker-compose up -d
```

Acesse o container do php, e rode o composer:

```shell
$ docker-compose exec php bash
$ composer install
```

## Uso

O projeto pode ser acessado pela url `http://localhost`
